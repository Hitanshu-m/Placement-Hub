# 🎓 Placement Hub

## 📌 About
This is a job placement web application built using React + Vite and Node.js. It connects companies and students in one platform. Companies can post job opportunities, and students can browse and apply based on their skills. The platform also includes a multilingual chatbot to help users easily navigate and understand features. The goal of this project is to simplify the hiring and placement process.
